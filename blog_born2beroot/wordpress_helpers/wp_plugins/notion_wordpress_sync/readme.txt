Sync content from Notion to WordPress.

== Description ==

Notion WP Sync allows you to connect your Notion workspace to WordPress and sync content automatically or manually. This plugin makes it easy to write in Notion and publish to WordPress.

Features:

* Connect to your Notion workspace via the Notion API
* Select which Notion pages/databases to sync
* Map Notion content to WordPress posts/pages
* Automatically sync content on a schedule or manually
* Preserve formatting, code blocks, and images

== Installation ==

1. Upload the `notion-wp-sync` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Notion WP Sync to configure the plugin
4. Enter your Notion API key and configure content mappings

== Frequently Asked Questions ==

= How do I get a Notion API key? =

You need to create an integration in Notion. Go to https://www.notion.so/my-integrations and create a new integration. Make sure to give it access to the databases and pages you want to sync.

= How often does the plugin sync content? =

You can choose between hourly, twice daily, or daily sync intervals. You can also manually sync content at any time.

= Can I sync images from Notion to WordPress? =

Yes, the plugin will download images from Notion and add them to your WordPress media library.
